Bem-vindo à seção dedicada aos meus projetos de desenvolvimento web! Este espaço marca o início da minha jornada no Front-End, onde transformo ideias em interfaces digitais interativas e funcionais.

Aqui você encontrará a evolução dos meus estudos, desde conceitos básicos de estruturação até layouts mais complexos e modernos.

🛠️ O que eu exploro nestes projetos:
Estruturação Semântica: Uso de HTML5 para garantir que a web seja acessível e bem interpretada pelos motores de busca (SEO).

Estilização Avançada: Domínio de CSS3 para criação de layouts, trabalhando com Flexbox, Grid e animações.

Responsividade: Desenvolvimento de sites que se adaptam perfeitamente a qualquer tamanho de tela, do celular ao desktop (Mobile First).

Hospedagem e Deploy: Experiência em colocar projetos online utilizando ferramentas como GitHub Pages e Bohr.io.
